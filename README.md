# danielbob0201-png's Academic Website
**Personal academic and research portfolio hosted on GitHub Pages.**

This site is built from the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template.

## Getting Started

1. Clone the repository: `git clone https://github.com/danielbob0201-png/danielbob0201-png.github.io.git`
1. Edit site-wide configuration in `_config.yml` — update `title`, `name`, `description`, `author`, `url`, and `repository`.
1. Add your content to the `_pages/`, `_posts/`, `_publications/`, `_talks/`, and `_teaching/` directories.
1. Push changes to the `main` branch to publish to https://danielbob0201-png.github.io

## Running locally

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The local server will be available at http://localhost:4000.

## Site contents

- **Home / About** — `_pages/about.md`
- **Publications** — `_publications/`
- **Talks** — `_talks/`
- **Teaching** — `_teaching/`
- **Blog / News** — `_posts/`
