## About

I am a first-year PhD candidate at the [HKUST NLP Group](https://hkust-nlp.github.io/), advised by [Prof. Junxian He](https://jkjung.com/). I received my B.Eng. from [Shanghai Jiao Tong University (SJTU)](https://www.sjtu.edu.cn/) in June 2024.

### Research Interests
- **LLM Reasoning and Reinforcement Learning** — Developing methods to improve reasoning capabilities in large language models and leveraging reinforcement learning for more reliable AI systems
- **Hallucination in Vision-Language Models (VLM)** — Understanding and mitigating when VLMs generate factually inconsistent outputs
- **LLM Truthfulness and Interpretability** — Exploring internal mechanisms of LLMs that govern truthfulness and making models more interpretable

## Getting Started

1. Clone the repository: `git clone https://github.com/danielbob0201-png/bugmaker00.github.io.git`
1. Edit site-wide configuration in `_config.yml` — update `title`, `name`, `description`, `author`, `url`, and `repository`.
1. Add your content to the `_pages/`, `_posts/`, `_publications/`, `_talks/`, and `_teaching/` directories.
1. Push changes to the `master` branch to publish to https://bugmaker00.github.io

## Running locally

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

The local server will be available at http://localhost:4000.
