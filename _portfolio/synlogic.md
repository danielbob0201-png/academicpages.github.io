### Synopsis

SynLogic is a framework for synthesizing large-scale, high-quality, and verifiable logical reasoning data. It employs a multi-stage generation pipeline combining symbolic logic synthesis with natural language generation to produce reasoning chains with verified correctness.

**Key contributions:**
- A novel data synthesis framework targeting verifiable logical reasoning
- A systematic evaluation showing improvements in LLM reasoning benchmarks
- Open-sourced dataset and code for research use
