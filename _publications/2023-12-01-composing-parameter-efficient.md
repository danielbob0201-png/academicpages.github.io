**Composing Parameter-Efficient Modules with Arithmetic Operations**

Jinghan Zhang, Shiqi Chen, Junteng Liu, Junxian He

NeurIPS 2023

[Paper](https://papers.neurips.cc/paper_files/paper/2023/hash/xxx.html)

## Abstract

We propose a method for composing parameter-efficient fine-tuned modules using arithmetic operations, enabling efficient multi-task learning and task composition without retraining or storing separate parameter sets for each task combination.
